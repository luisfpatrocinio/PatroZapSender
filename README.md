# WhatsApp Auto Messenger

A modular Python automation script designed to send bulk personalized WhatsApp Web messages using data parsed from a CSV file. 

## Features
* Parses detailed customer address data from CSV files using `pandas`.
* Generates custom, personalized message templates.
* Automates the browser interaction with WhatsApp Web using `pyautogui`.

## Prerequisites
* Python 3.x
* A default web browser logged into WhatsApp Web.
