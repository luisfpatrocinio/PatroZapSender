# PatroZapSender

A lightweight Python automation tool designed to send bulk personalized WhatsApp Web messages using customer data parsed from a CSV file.

## Features

- Parses detailed address and customer data from CSV files using `pandas`.
- Generates custom, personalized message templates dynamically.
- Automates browser interaction with WhatsApp Web using `pyautogui`.

## Prerequisites

- Python 3.x
- A default web browser logged into WhatsApp Web.

## Installation

1. Clone this repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
